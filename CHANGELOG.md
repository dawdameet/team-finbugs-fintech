# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Added
- Initial release of FinBugs Analytics Platform
- Market Sentiment Analyzer with VADER integration
- AI Portfolio Optimizer with MPT implementation
- Stock Price Predictor with multiple ML models (Linear Regression, Random Forest, LSTM)
- Monte Carlo simulation service in C++
- Pair trading backtest engine in C++
- REST API with FastAPI
- Docker containerization support
- CI/CD pipeline with GitHub Actions
- Comprehensive test suite
- Documentation and examples

### Features
- Real-time market sentiment analysis
- Efficient frontier calculation
- Value at Risk (VaR) computation
- Beta and Sharpe ratio calculations
- ML-based asset clustering
- Technical indicators (RSI, MACD, Bollinger Bands)
- GARCH volatility modeling
- Jump-diffusion processes
- Mean-reversion trading strategy

### Infrastructure
- PostgreSQL database integration
- Redis caching layer
- Prometheus metrics
- Docker Compose setup
- Automated testing
- Code coverage reporting
